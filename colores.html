Creación y uso de funciones personalizadas

📚 Glosario de Funciones

imread

Funciòn que permite leer y abrir una imagen.

imshot

Mostrar la imagen en una ventana

Parámetros:

base - Ancho del rectángulo (número)

altura - Alto del rectángulo (número)

Retorna:

El área del rectángulo (número)

es_par(numero)

Verifica si un número es par utilizando el operador módulo (%).

Parámetros:

numero - Número entero a verificar

Retorna:

True si el número es par, False si es impar (booleano)

saludar(nombre, mensaje="Hola")

Genera un saludo personalizado combinando un mensaje con un nombre.

Parámetros:

nombre - Nombre de la persona a saludar (texto)

mensaje - Mensaje de saludo (texto, opcional, por defecto "Hola")

Retorna:

Cadena de texto con el saludo completo
